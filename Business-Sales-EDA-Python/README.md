# Business Sales Analysis (EDA using Python)

## Objective
To analyze business sales data and identify trends, patterns, and key insights using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Steps Performed
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Sales trend analysis
- Visualization of key metrics
- Insight generation

## Key Insights
- Identified top-performing products and regions
- Observed seasonal sales patterns
- Found factors impacting sales growth

## Files in this Project
- `business_sales_eda.ipynb` – Complete Python EDA code
- `business_sales_dataset.csv` – Dataset used
- `Business_Sales_EDA_Report.pptx` – Final insights presentation
