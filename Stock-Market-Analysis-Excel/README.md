# Stock Market Analysis (5 Years Data)

## Objective
Analyze 5 years of stock market data using Excel
to understand trends, growth, and performance.

## Tools Used
- Microsoft Excel

## Steps Performed
- Data cleaning and formatting
- Year-wise trend analysis
- CAGR and growth calculations
- Visualization using Excel charts

## Key Insights
- Identified long-term growth trends
- Compared yearly performance variations
